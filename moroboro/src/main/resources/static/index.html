How to get YouTube credentials?
  1. Go to the Google Cloud Console.
  2. Create a new project and search for YouTube Data API v3. Enable it.
  3. Click on Credentials -> Create Credentials -> API Key. Copy the key and paste it above.
  4. To find your Channel ID:
    • Go to YouTube -> Login -> Settings -> Advanced Settings.
    • Copy the Channel ID (starts with UC).

Security Note: Your credentials entered in settings are saved only in your local browser storage (localStorage) and never uploaded to public clouds.
